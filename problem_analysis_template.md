### 🚨 常见陷阱与易错点

#### 逻辑陷阱
- [ ] **边界条件遗漏** - 空输入、单元素、重复元素
- [ ] **循环条件错误** - off-by-one错误
- [ ] **返回值错误** - 返回索引vs值、多个答案vs单个

#### 实现细节
- [ ] **数组越界** - 访问不存在的索引
- [ ] **整型溢出** - 大数运算
- [ ] **指针操作** - 空指针、野指针

#### 复杂度陷阱
- [ ] **时间复杂度** - 嵌套循环导致超时
- [ ] **空间复杂度** - 额外空间使用不当
